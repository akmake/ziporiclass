# Zipori Class - מערכת ניהול הזמנות ותפעול מלונות

מערכת CRM ו-PMS (Property Management System) מתקדמת לניהול רשת מלונות "ציפורי קלאס".
המערכת מספקת פתרון מקצה לקצה: החל משלב קליטת הליד והצעת המחיר, ועד לניהול התפעול השוטף, ניקיון החדרים והתחזוקה.

הפרויקט בנוי בארכיטקטורת Monorepo ומכיל:
- **Client**: אפליקציית React מודרנית עם Vite, TailwindCSS (shadcn/ui), Zustand לניהול מצב, React Query לסנכרון נתונים ו-Framer Motion לאנימציות.
- **Server**: שרת Express חזק עם Node.js, מסד נתונים MongoDB, אימות מאובטח (JWT + CSRF), ומערכת התראות (Web Push).

---

## ✨ פיצ'רים מרכזיים

### 🏨 ניהול הזמנות ומכירות (CRM)
- **ניהול לידים:** קליטה אוטומטית של פניות (Webhooks), סינון, סטטוסים ומעקב.
- **הזמנות:** יצירת הזמנות מורכבות, ניהול מחירונים דינמיים, חישוב מחירים אוטומטי לפי הרכב אורחים.
- **הצעות מחיר:** הפקת דפי נחיתה להצעת מחיר וייצוא ל-PDF לשליחה ללקוח.
- **מחשבונים:** לוגיקה עסקית חכמה לחישוב עלויות, תוספות והנחות.

### 🧹 תפעול ואחזקה (Operations)
- **תמונת מצב חדרים (Live):** דשבורד בזמן אמת המציג חדרים נקיים, מלוכלכים או בתקלה.
- **סידור עבודה יומי:** הקצאת משימות למחר (Checkout / Stayover).
- **ניהול משימות:** צ'ק ליסטים לניקיון, דיווח תקלות ותיקונים.
- **ממשק עובד:** מסך מותאם לעובדי שטח לדיווח ביצוע בזמן אמת.

### 🛡️ ניהול ובקרה (Admin)
- **דשבורד מנהלים:** סטטיסטיקות, גרפים, הכנסות וביצועי אנשי מכירות.
- **משתמשים והרשאות:** ניהול תפקידים (Admin, Sales, Maintenance) ובקרת גישה.
- **הגדרות מערכת:** ניהול מלונות, סוגי חדרים, מחירונים והודעות מערכת.
- **התראות:** מערכת Push Notifications לעדכון על לידים חדשים גם כשהדפדפן סגור.
